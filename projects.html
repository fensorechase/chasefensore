Simpson's Rule: Automate IR Data Integration

Technologies: Python, pandas, numpy, scipy

Description: I wrote this python script to automate integration of infrared spectrometry data. Area under the curve(AUC) of an IR spectrum is often useful becuase it is proportional to the concentration of absorbing moieties. However, an IR spectrometer outputs hundreds of files, making it tedious to calculate AUC for all csv files from a group of experiements. This program accespts user input for lower and upper integration limits, parses through a given folder of .csv files, and finds AUC for each input file using Simpson's rule. The AUC for each input file is written to a corresponding otuptut file.

Investment Events Webscraper

Technologies: Python, GCal API, beautiful soup(bs4)

Description: The purpose of this project was to streamline sourcing of relevant investment events for startups. I built a python script to write events to your Google Calendar using GCal API. The command line utility takes a input event URL, then parses with BeautifulSoup. Then, OAuth2.0 is used to verify the user's GCal account, and the event is automatically added to the user's GCal.
